require('flatiron');

An initiative to build a collection of decoupled tools with the same standard of quality and performance that you would expect from anything built by Nodejitsu.

A full-stack web application development framework which packages these tools together to make isomorphic and stream-based application development easier.

Define: Isomorphic Code

Code is said to be Isomorphic if it is possible to execute its API both on the server and in a browser.

Overwhelmed?

There currently 15 projects in the Flatiron initiative. Don't worry if this seems like a lot to take in at once. It's what we expected.

var flatiron = require('flatiron'), app = flatiron.app; app.use(flatiron.plugins.cli, { dir: __dirname }); app.cmd('app start', function () { app.prompt.get('name', function (err, name) { console.dir(arguments); }) }) app.init(function () { app.start(); });

Hello World (http)

var fs = require('fs'), favicon = require('./favicon'), flatiron = require('flatiron'), app = flatiron.app; app.use(flatiron.plugins.http, { before: [favicon] }); app.router.get(/foo/, function () { this.res.writeHead(200, { 'Content-Type': 'text/plain' }) this.res.end('hello world\n'); }); app.router.post(/foo/, { stream: true }, function () { var req = this.req, res = this.res, writeStream; writeStream = fs.createWriteStream(Date.now() + '-foo.txt'); req.pipe(writeStream); writeStream.on('close', function () { res.writeHead(200, { 'Content-Type': 'text/plain' }); res.end('wrote to a stream!'); }); }); app.start(function () { console.log('flatiron with http running on 8080'); });
